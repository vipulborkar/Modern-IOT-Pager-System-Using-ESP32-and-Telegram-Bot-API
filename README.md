# Modern-IOT-Pager-System-Using-ESP32-and-Telegram-Bot-API
The Modern IoT Pager System is an ESP32-based wireless device that receives real-time Telegram messages, displays them on an LCD, and supports instant hardware-button replies. It features message logging, audio alerts, and automatic Wi-Fi reconnection.
